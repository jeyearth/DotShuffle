# DotShuffle

心に響いた言葉を簡単に記録し、ランダムに振り返ることができる iOS アプリです。シンプルな操作性と直感的なデザインを重視し、ユーザーが気軽に日々の言葉を蓄積できるよう設計しました。

「DotShuffle」をApp Storeで：
https://apps.apple.com/jp/app/dotshuffle/id6498897832?platform=iphone
